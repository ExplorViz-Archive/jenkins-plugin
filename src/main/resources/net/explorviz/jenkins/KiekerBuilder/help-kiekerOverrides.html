Specifies properties to override the automatically generated values for the kieker monitoring configuration, in the java properties syntax: 

# comment
name1=value1
name2=value2

Not all properties can be overriden when proper function of the plugin should be maintained. Most importantly, when overriding kieker.monitoring.writer and kieker.monitoring.writer.filesystem.FileWriter.customStoragePath the plugin will no longer be able to find the records.
