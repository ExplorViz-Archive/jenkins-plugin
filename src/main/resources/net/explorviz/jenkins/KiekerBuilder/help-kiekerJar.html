Path to the kieker jar used as javaagent for the instrumentation, relative to the workspace directory. Make sure this file exists at the time of this build step, i.e. by shipping it in your repository.

Note: Only aspectj variants are supported for now. Also make sure the version you're trying to use is recent enough to have the kieker.monitoring.writer.filesystem.FileWriter writer.

Leave empty to use the bundled Kieker 1.14-SNAPSHOT-aspectj.
